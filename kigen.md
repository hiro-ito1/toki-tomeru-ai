# 技術的突然変異の法則 ― 起源

---

## この法則が生まれるまで

高校に入った時、私はテニスの初心者だった。
友達に教わりながら、ひたすらイメトレだけをやっていた。

大学2年の春。先輩に一言言われた。
「もっと簡単に決めろ。」

その言葉の意味がわからなかった。
「もっと簡単に決めるとは？どういうこと？」
その問いから、考え始めた。

このちょっと後から、頭が鋭くなっていった。
いろいろな仕組みが見え始めた。
答えノートもその一つ。ユメのシステムの考えの種も、ここにある。

答えノートと試合ノートが先に生まれた。
イメトレは上達のためだと思って一生懸命やっていた。
でも当時の私は、「定着＝無意識でできるようになるまでやる」という概念に、まだ気づいていなかった。

フォアをやればバックが崩れる。
バックをやればフォアが崩れる。
崩れた方を治しに行けば、また別が崩れる。
どちらも中途半端なまま、永遠に繰り返す。

> ※なぜそうなるのか。人間が同時に意識できるのは1つ。定着前の技術はまだ「考えながらやっている状態」。別の技術に意識が移った瞬間、定着していない技術はできなくなる。

当時の私自身が、そのループの中にいた。

大学2年から4年の初旬まで、大学まで徒歩15分＋電車1時間。
帰りの電車では毎日ノートを書いた。仮定、推論、振り返り。
真っ暗な帰り道を歩きながら素振りをしていた。
後ろから来た自転車に当てそうになったこともある。
危険人物と間違われたこともあった。

---

## 震えた瞬間

ある日、一人でボールを手で上に上げて、それを打っていた。

その時、確信した。

**「リアルと同じようにイメトレができれば、人の10倍練習できる。上手くなるやん！」**

体が震えた。
その感覚は、今でも憶えている。

これで最後のピースが繋がった。
「定着＝イメトレでできる」。
技術的突然変異の法則が、完成した瞬間だった。

イメトレなしでは、この法則は全然うまくいかない。
実際の練習量だけでは、超強敵に勝つには少なすぎるから。

## 沿革

| 時期 | 出来事 |
|---|---|
| 高校入学時 | 初心者としてテニスを始める。友達に教わりながら、イメトレだけをやっていた。 |
| 大学2年 春 | 先輩の「もっと簡単に決めろ」という一言から考え始める。頭が鋭くなり、仕組みができ始める。答えノート・試合ノートが生まれる。 |
| 大学2年〜4年初旬 | 帰りの電車で毎日ノートを書き、帰り道で素振りを続ける。 |
| 大学2年 夏 | 九州学生ダブルス 準優勝。 |
| 大学3年 春 | 九州学生ダブルス 優勝。 |
| 大学3年 夏 | 九州学生シングルス 優勝。 |
| 大学3年 冬（引退数か月前） | 完全なるイメトレを発見。技術的突然変異の法則が完成。 |
| 大学4年になる春 | 九州学生シングルス 連覇。 |
| 2015年10月 | 「突然変異する為の進化論」を紙にして配布。技術的突然変異の法則が名前と形を持った起点。 |
| その後 | 練習会は2期生・数期生と続く。現在は「時を止める指導法」のOSに統合。 |
